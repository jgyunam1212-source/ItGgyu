**AI-powered Windows BSOD (Blue Screen of Death) Dump Analyzer using Gemini API.** (Gemini API를 활용한 AI 기반 윈도우 블루스크린 덤프 분석기)

Tired of using complex WinDbg to analyze `.dmp` files?  
ItGgyu BSOD Analyzer AI PRO automatically analyzes Windows crash dumps and Event Viewer logs using Google's Gemini AI to provide easy-to-understand causes and solutions.

어렵고 복잡한 WinDbg 대신, AI가 블루스크린 덤프 파일(.dmp)과 이벤트 뷰어 로그를 분석하여 정확한 원인과 해결책을 가장 알기 쉽게 알려줍니다.

## ✨ Features (주요 기능)

* **🤖 Smart AI Analysis:** Analyzes minidump files and finds the exact faulty driver/hardware using Gemini AI.
* **📁 Batch Processing:** Support for single file and multiple dump folder analysis.
* **📋 Event Log Integration:** Analyzes Windows Event Viewer logs together for precise diagnosis.
* **🌐 Global Support:** Multi-language output support (English, Korean, Japanese, Chinese).
* **🎨 Cyber-Specialist UI:** Sleek and modern dark theme tailored for professionals.
* **⚡ Portable:** No installation required. Just run the `.exe` file.

## 🚀 How to Use (사용 방법)

1. Download the latest portable version from the [Releases](../../releases) tab.
2. Get your free Gemini API Key from [Google AI Studio](https://aistudio.google.com/app/apikey).
3. Run `BlueScreenAnalyzer.exe`.
4. Go to **Settings**, paste your API Key, and select your preferred language.
5. Click **Select File** to load a `.dmp` file (usually located in `C:\Windows\Minidump`).
6. Get your AI-generated crash report in seconds!

(한국어)
1. 우측 [Releases] 탭에서 최신 포터블 버전을 다운로드합니다.
2. [Google AI Studio]에서 무료 Gemini API 키를 발급받습니다.
3. 프로그램을 실행하고 **설정** 메뉴에 API 키를 입력합니다.
4. 분석할 덤프 파일(`C:\Windows\Minidump`)을 선택하면 AI가 즉시 원인을 분석합니다.
